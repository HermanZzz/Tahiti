## Getting Started

1. Start Server

	Run `Server` configuration in Intellij IDEA, or:
	
	```bash
	cd Tahiti/Server
	mvn exec:exec
	```

2. Start Client

	Run `Client` configuration in Intellij IDEA, or:
	
	```bash
	cd Tahiti/Client
	mvn exec:exec
	```

## Available Commands for Client

#### login

- username
- password

#### ping

- payload
